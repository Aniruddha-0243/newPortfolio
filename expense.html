Personal Expense Tracker

The Personal Expense Tracker is a web application designed to help users manage and track their daily expenses efficiently. It provides an intuitive interface to add, categorize, and visualize expenses, helping users maintain budgets and gain insights into their spending habits.

Features

  • Track daily expenses with categories
  • View summaries and charts of spending
  • Set monthly budgets and alerts
  • Responsive and user-friendly interface
  • Data persistence using local storage
