Calculator

A web-based scientific calculator with a clean interface and functionality for common math operations.

Features

  • Basic and scientific calculations
  • Responsive and user-friendly interface
  • Keyboard and button input support
  • History of calculations
  • Clear and reset functionality

Technologies Used

  • HTML5, CSS3, JavaScript
  • Bootstrap 5 for responsive design
  • Bootstrap Icons for UI elements

Challenges Faced

Ensuring accurate scientific calculations and handling edge cases in user input were challenging. Implementing keyboard support required careful event handling to avoid conflicts.

Future Improvements

Adding more advanced mathematical functions, improving the UI with themes, and integrating a history export feature are planned for future updates.
